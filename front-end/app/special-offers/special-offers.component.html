Loyalty Scheme

With every order you place, you'll receive a unique voucher code worth 10% of the order. You can use this voucher code any time within 28 days to receive an additional discount on your next order.

For example, if your order costs you £20, you'll receive a voucher code that will give you £2 off your next order placed within 28 days.

The voucher code is contained in the order confirmation email, which we send once we've confirmed receipt of your order.
